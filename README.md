# KiCAD .gitignore generator

Adds a button in the toolbar that automatically generates a .gitignore for your KiCAD project!

![screenshot of toolbar](/assets/toolbar.png)

_(it's the git icon on the right)_

File based on the gitignore repository: https://github.com/github/gitignore

For some reason I literally could not find a plugin that does this.

---

### Install

Add the following URL to your KiCAD repositories:

```
https://dari.zone/kicad/repository.json
```

Search for "gitignore"

Click install!
